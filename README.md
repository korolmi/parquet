# Parquet: формат файла и метаданные

Поддерживающие материалы и код.

## Thrift вариант просмотра метаданных

* `parquet_meta_viewer.ipynb`: полная версия ноутбука, упоминающегося и цитируемого в статьях про thrift вариант
  * ноутбук содержит output, позволяющий понять - что происходило во время его реального выполнения 
* `ttypes.py`: сгенерированнаый THRIFT компилятором по оригинальной спецификации python модуль (как есть - не менял)
* `pq_thrift_utils.py`: функции, облегчающие работу с сгенерированным модулем
  * здесь функции `decodeObject()`, `getLevelStr()` и другие 
* `extra_docs.yaml`: дополнительные описания объектов, используется в функции `showExtra()` модуля `pq_thrift_utils.py`

## DuckDB вариант просмотра метаданных  

* `parquet_play.ipynb`: ноутбук, позволяющий интерактивно анализировать метаданные
* `viz_tools.py`: функции для визуального выбора объектов визуализации (row group, column, bloom filter)
  * ноутбук `parquet_play.ipynb` "опирается" на этот модуль
